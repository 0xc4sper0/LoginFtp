# LoginFtp
Tool Scan anonymous Tester login For FTP


1- Usage: python3 ftplogin.py <filename-domain-list> [port]

2- If the port is variable from 21, enter it.

3- Scan nmap ports: nmap -Pn <Domain/Ip>
